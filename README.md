# Séance 20 - Audit technique client

Support web de formation pour transformer un R1 client en audit technique actionnable :

- transcription complète de la vidéo ;
- extraction des moments importants ;
- analyse du besoin client ;
- recherche via Playwright MCP ;
- construction d'une recommandation IA déduite du call ;
- présentation HTML responsive avec prompts copiables ;
- publication GitHub Pages.

La vidéo source reste locale et n'est pas publiée.
